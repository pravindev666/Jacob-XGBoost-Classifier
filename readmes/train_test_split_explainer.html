Why can't we train up to today?
The Exam Cheat Sheet Problem
Imagine a student who memorises the answer key before the exam. They score 100% on that exam but fail the next one completely. Training your model on data right up to today is the same thing — it "memorises" recent patterns and scores fake 90%+ accuracy. But when tomorrow comes, it fails because it never truly learned to predict.

Train (2015-2021)
The model STUDIES this data. It learns: "When RSI drops below 30 AND VIX spikes above 20, Nifty usually bounces next day." ~1,500 trading days of patterns.
Validate (2022-2023)
Used during training to check if the model is memorising (overfitting). XGBoost early-stopping watches this — it stops when val accuracy stops improving.
Test (2024-2025)
The model has NEVER seen this data. This gives the TRUE accuracy — if it's 58-62% here, that's what you'll get in live trading. This is your report card.
What happens AFTER the test set (2025-now)?
This is your LIVE trading zone. The model predicts on today's data in real-time. Every day the model makes a fresh prediction on the new row — it was never trained on this data, so the accuracy you saw on the test set is what you should expect here.

Why this fails
The model scores 85%+ accuracy on its own training data (it memorised it). But when tomorrow actually arrives, accuracy drops to 50% — coin flip level. You can't test it on data it already studied. You need data it has NEVER seen to get a real accuracy number. That's why we keep 2024–2025 completely locked away during training.
